== Description ==
**Woocommerce AlsatPardakht Gateway** این افزونه شما را قادر می سازد تا براحتی اقدام به ایجاد درگاه پرداخت اینترنتی برای پرداخت های محصولات ووکامرس کنید

== Installation ==
قبل از همه در وبسایت آل‌سات پرداخت ثبت نام کرده و پس از درخواست درگاه و ارسال مدارک افزونه را دانلود، نصب و فعال سازی افزونه از قسمت پیکربندی ووکامرس -> پرداخت ها -> پرداخت امن آل‌سات پرداخت -> مدیریت تکمیل کنید، سپس میتوانید 
تنظیمات افزونه را به شکل دلخواه خود تغییر دهید
